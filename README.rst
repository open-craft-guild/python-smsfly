python-smsfly
-------------

SMS-Fly gateway API Python package

Install it
##########

.. code-block:: sh

    pip install SMSFly

Testing
#######

.. code-block:: shell

    pip install tox --user
    tox

Usage
#####

.. code-block:: python
   :number-lines:

    from smsfly import SMSFlyAPI
    api = SMSFlyAPI(account_id='3801234567', account_pass='qwerty')
    api.getbalance()
